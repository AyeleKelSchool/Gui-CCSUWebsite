# Gui-CCSUWebsite
This is a Graphical User Interface program that display different information regarding CCSU such as the name of the buildings, the Fall 2025 Calendar and etc...
The program is also able to take a image, preferably one with the same name as listed in the code, and then transform it white side into transparent and display it as a logo within the window while also display the image itself on the top right right while running.
User can switch between the Calendar, to the name of the buildings and Faculties, displaying the School of business and the department of Management Information Systems(MIS).
